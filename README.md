### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### SERVER
Laravel 10x - PHP 8.1 
API - 127.0.0.1/api/..

```
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=vodokachka
DB_USERNAME=root
DB_PASSWORD=
```
### FOR RUN 
1. Make migration 
```
php artisan migrate or export DB
```
2. Start APP - php artisan serve
### Run server
```
php artisan serve
```
